# Smart Green Infrastructure (RehabAi)

Explorations for a smart green infrastructure concept, combining pose estimation and sequence modeling ideas.

## Contents
- **simple mediapipe+numpy.ipynb** – quick prototype for pose extraction and processing.
- **Ai assisted St-gcn implementation (2018 paper).ipynb** – experimental implementation inspired by the ST-GCN paper for action/pose modeling.

## Skills highlighted
- Computer vision prototyping
- Working with human pose data
- Translating research papers into code experiments
