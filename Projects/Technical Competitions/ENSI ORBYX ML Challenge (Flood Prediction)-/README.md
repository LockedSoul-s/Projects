# ENSI ORBYX ML Challenge – Flood Prediction

Project notebooks built for a flood prediction challenge. The work reflects iterative modeling from baseline approaches to a more refined pipeline.

## Contents
- **baseline-informed-approach.ipynb** – a baseline model with core preprocessing and feature setup.
- **winning pipeline.ipynb** – a more advanced pipeline with improved modeling and evaluation.
- **ExperimentalBroken.zip** – archived experiments and rough ideas that didn’t make it into the final pipeline.

## Skills highlighted
- Structured notebook workflow
- Iterative improvement and experiment tracking
- Practical model evaluation
