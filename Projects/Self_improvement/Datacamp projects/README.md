# DataCamp Projects

These are complete DataCamp project bundles I completed to practice real-world data workflows. Each zip contains the project instructions, datasets, and my notebook solutions.

## Project list
- **Fuel_cost forecasting with sarimax.zip** – time-series forecasting using SARIMAX with trend/seasonality handling.
- **Londonium!.zip** – data wrangling and exploratory analysis on a London-themed dataset.
- **Developing multi-input model for OCR.zip** – multi-input neural network approach for optical character recognition.
- **Hypothesis Testing with Men's and Women's Soccer Matches.zip** – statistical tests comparing groups and interpreting p-values.
- **Analyzing Crime in Los Angeles.zip** – exploratory analysis of crime patterns over time and location.
- **SAMPLE EXAM Data scientist associate practical.zip** – end-to-end case study covering preprocessing, modeling, and reporting.

## Skills highlighted
- Data cleaning and feature engineering
- Statistical reasoning
- Time-series modeling
- Introductory deep learning
